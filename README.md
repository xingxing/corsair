## Usage 

`` 
    cd ~ && \
    git clone git@github.com:xingxing/corsair.git && \
    echo '(load-file "~/.emacs.d/init.el")' > ~/.emacs && \
    mv ~/corsair.git ~/.emacs.d && \
    cd ~/.emacs.d && git submodule init && \
    git submodule update 
``
